Computer Components

  • CPU (Central Processing Unit) - The CPU, or "brain" of the computer, executes instructions and processes data for all tasks.
  • Motherboard - The motherboard connects all components and facilitates communication between them.
  • RAM (Memory) - RAM provides temporary storage for data in use, allowing quick access for the CPU.
  • Storage (HDD/SSD) - Storage holds all data and files, including the operating system, even when the computer is off.
  • Power Supply Unit (PSU) - The PSU converts and distributes power to all computer parts, ensuring stable operation.
  • GPU (Graphics Processing Unit) - The GPU handles graphics rendering, essential for gaming, video editing, and 3D tasks.
  • Input and Output Devices (I/O) - I/O devices allow users to interact with the computer and view results.
